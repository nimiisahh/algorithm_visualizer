Algorithm Efficiency

Merge Sort is a highly efficient, stable, and comparison-based sorting algorithm. It uses a divide-and-conquer strategy to sort the array.

- Best Case: O(n log n)
- Average Case: O(n log n)
- Worst Case: O(n log n)

Merge Sort divides the array into halves, recursively sorts them, and then merges the sorted halves into a single sorted array.
