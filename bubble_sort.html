Algorithm Efficiency

Bubble Sort is a simple but inefficient sorting algorithm. In both the worst and average cases, it has a time complexity of O(n²), which means the execution time grows quadratically with the size of the array.

- Best Case: O(n) — when the array is already sorted
- Average/Worst Case: O(n²) — many comparisons and swaps are needed

Due to its inefficiency, Bubble Sort is mainly used for educational purposes, to demonstrate the concept of sorting via repeated comparisons and swaps.
