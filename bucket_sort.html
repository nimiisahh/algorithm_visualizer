Algorithm Efficiency

Bucket Sort is an efficient algorithm when the input is uniformly distributed over a range. It works by dividing elements into buckets and then sorting each bucket individually.

Time Complexity: Θ(n + k)
Space Complexity: Θ(k)
where n is the number of elements and k is the range of input values (assuming values in {0,...,k-1}).

The implementation assumes all elements in the array are integers in the range {0, ..., k-1}.
