Algorithm: Selection

Selection is used to find the k-th smallest element in an unsorted array. It works by choosing a pivot (like in QuickSort), and partitioning the array:

  • If k equals the pivot's position, return it.
  • If k is smaller, recurse into the left partition.
  • If k is larger, recurse into the right partition.

Time Complexity:
Best Case: Θ(n)
Average Case: Θ(n)
Worst Case: O(n²)
