Radix Sort

10

Algorithm:

Radix Sort processes each digit of the numbers starting from the least significant digit (LSD). It repeatedly applies a stable sort (like Bucket Sort) based on the current digit. The process continues for all digit places until the list is sorted.

Efficiency Analysis:

  • Time Complexity: Θ(d·(n + k))
  • Space Complexity: Θ(n + k)

Note: d is the number of digits, n is the number of elements, and k is the range of digit values (typically 10 in base-10).
