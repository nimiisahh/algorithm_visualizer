Sorted Array

Algorithm:

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It first builds a max heap from the input array, then repeatedly extracts the maximum element from the heap and places it at the end of the array.

Efficiency Analysis:

  • Time Complexity: Θ(n·log n)

Note: Heap Sort is not a stable sort, but it is in-place and guarantees O(n log n) time regardless of input.
